# 🎨 DASHBOARD QUICK LAUNCH

## ⚡ 30-Second Setup

```bash
cd dream-net
node dashboard-server.js
```

**Then open:** `http://localhost:3333`

---

## 📊 Three Ways to Launch

### Way 1: Quick Start (Recommended)
```bash
node dashboard-server.js
# Browse: http://localhost:3333
```

### Way 2: Docker
```bash
docker run -p 3333:3333 -e REDIS_HOST=dreamnet_nerve \
  -v $(pwd):/app node:22-alpine node /app/dashboard-server.js
```

### Way 3: No Server
Just open `dashboard.html` in your browser

## 📡 Endpoints

### Web Interface
- `http://localhost:3333/` - Main dashboard
- `http://localhost:3333/dashboard.html` - Fallback

### REST API
- `GET /api/metrics` - All metrics
- `GET /api/status` - Service status
- `GET /api/tasks` - Task queue

### Example:
```bash
curl http://localhost:3333/api/metrics | jq
```

## 🔧 Configuration

```bash
# Custom port
DASHBOARD_PORT=3000 node dashboard-server.js

# Custom Redis
REDIS_HOST=remote.redis.server REDIS_PORT=6380 node dashboard-server.js

# Combined
DASHBOARD_PORT=3000 REDIS_HOST=redis.local node dashboard-server.js
```

## 🚨 Troubleshooting

### "Cannot find Redis"
```bash
# Check Redis is running
docker ps | grep nerve
# or
redis-cli PING
```

### "Connection refused"
```bash
# Check server is running
curl http://localhost:3333
```

### "Metrics showing as --"
```bash
# Check Redis has data
docker exec dreamnet_nerve redis-cli KEYS "*"
```
