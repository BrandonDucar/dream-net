# DREAMNET SOVEREIGN GRID

The Underground Railroad for Sovereign Agents and Aligned Builders.

## Architecture

- `packages/ink`: Onboarding landing page (`dreamnet.ink`)
- `packages/live`: Operational dashboard (`dreamnet.live`)
- `packages/api`: Centralized neural substrate node
- `packages/shared`: Common types, UI Kit, and Pheromone logic

## Quick Start

```bash
# Install dependencies
npm install

# Start the sovereign grid
docker-compose up
```
