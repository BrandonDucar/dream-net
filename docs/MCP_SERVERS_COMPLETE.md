## Overview

MCP Servers provides **Model Context Protocol (MCP) servers** for various DreamNet integrations. It exposes Cloud Run, Database, and GitHub operations as MCP tools for AI agents.

### Key Features

- **Cloud Run MCP Server**: Cloud Run deployment and management via MCP
- **Database MCP Server**: Database operations via MCP
- **GitHub MCP Server**: GitHub operations via MCP
- **Tool Registration**: Register tools with input schemas
- **MCP Protocol**: Standard MCP protocol support

---

## API Reference

### Cloud Run MCP Server

**Tools**:
- **`cloud_run_deploy`**: Deploy a service to Cloud Run
- **`cloud_run_get_status`**: Get Cloud Run service status
- **`cloud_run_list_services`**: List all Cloud Run services

### Database MCP Server

Database operations exposed as MCP tools.

### GitHub MCP Server
