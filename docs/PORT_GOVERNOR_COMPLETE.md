## Overview

Port Governor provides **governed ports** with roles, limits, and access control. It manages edge ports that connect external systems to DreamNet's internal clusters.

### Key Features

- **Port Management**: Register and manage edge ports
- **Role-Based Access**: Role-based access control for ports
- **Rate Limiting**: Rate limits and quotas per port
- **Access Control**: Control which clusters/tools ports can access
- **Port Wrapper**: Higher-order function to wrap ports with governance

---

## API Reference

### Main Exports

- **`types`**: Port types and interfaces
- **`ports`**: Port registry and management
- **`withPort`**: Port wrapper with governance
