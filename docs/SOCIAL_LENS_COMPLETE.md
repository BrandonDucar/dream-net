## Overview

Social Lens provides **Lens Protocol integration** for DreamNet. It enables interaction with Lens Protocol profiles, publications, and social graph.

### Key Features

- **Profile Management**: Access Lens Protocol profiles
- **Publication Management**: Create and manage publications
- **Social Graph**: Navigate Lens social graph
- **Protocol Integration**: Direct Lens Protocol API integration
