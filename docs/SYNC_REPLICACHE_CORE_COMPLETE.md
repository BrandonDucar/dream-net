## Overview

Sync Replicache Core provides **Replicache synchronization** for offline-first applications with conflict resolution and real-time sync.

### Key Features

- **Replicache Server**: Server-side Replicache implementation
- **Replicache Client**: Client-side Replicache implementation
- **Mutators**: Define data mutations
- **Sync**: Real-time synchronization
- **Conflict Resolution**: Handle sync conflicts
