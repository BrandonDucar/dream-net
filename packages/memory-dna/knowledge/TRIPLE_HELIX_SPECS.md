# Knowledge: Triple Helix Design Specifications

**Origin**: DreamForge / DreamNet Legacy (2025)
**Status**: INGESTED

## Core Strands

### 1. Intelligence (The Agent Strand)

* **Definition**: Autonomous cognitive agents (Nano-swarms) capable of planning, executing, and self-healing.
* **Reflex**: Sensing -> Local Reflex -> Homeostasis.

### 2. Sovereignty (The Financial Strand)

* **Definition**: On-chain autonomy and resource management.
* **Guardrails**: Emergency Cost Stop + 4337 Session TTLs + Economic Caps.
* **Engine**: The Dutch Oven (Arbitrage) + TypeX (Labor).

### 3. Distribution (The User Strand)

* **Definition**: Multi-platform delivery and viral engagement.
* **Channels**: OTT Streaming (Roku/FireTV), Farcaster Frames, Base Mini-Apps.
* **Orchestration**: Omnichannel testing via Browser-in-the-Loop.

## Merger Mandate

All future DreamNet developments must align with at least two strands of the helix to ensure maximum system resonance.
