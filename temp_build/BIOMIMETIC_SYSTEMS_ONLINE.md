## üêå Dream Snail - Know-All Win-All Privacy Layer

**Location**: `/snail`

Dream Snail is a comprehensive privacy system that:

- **Tracks Everything**: Records all user interactions and data flows
- **Verifiable Provenance**: Creates hash-chained trails for auditability
- **Privacy Controls**: Encryption, zero-knowledge proofs, analytics controls
- **Privacy Insights**: Automatically detects patterns, anomalies, and privacy risks
- **Trail Integrity**: Verifies hash chains to ensure data hasn't been tampered with

### Features

1. **Trail Recording**: Every event gets recorded with metadata
2. **Privacy Configuration**: User-controlled encryption, zero-knowledge, analytics
3. **Insights Engine**: Pattern detection, privacy alerts, recommendations
4. **Analytics Dashboard**: Privacy score, activity patterns, most active days
5. **Trail History**: Complete audit trail with privacy levels

### API Endpoints

- `GET /api/snail/trail` - Get user's trail
- `POST /api/snail/trail` - Record a trail event
- `GET /api/snail/privacy` - Get privacy config
- `PUT /api/snail/privacy` - Update privacy config
- `GET /api/snail/insights` - Get privacy insights
- `GET /api/snail/analytics` - Get analytics
- `GET /api/snail/verify` - Verify trail integrity
