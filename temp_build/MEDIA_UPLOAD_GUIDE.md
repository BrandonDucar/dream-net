You can now upload your Grok, Sora, YouTube music, and other media directly to DreamNet's Media Vault!

### How to Access

1. **Via Navigation**: Click "Upload Media" in the main navigation
2. **Direct URL**: Visit `https://dreamnet.ink/media-upload` (or `http://localhost:5173/media-upload` locally)

### What You Can Upload

- **Images**: JPG, PNG, GIF, WebP
- **Videos**: MP4, MOV, AVI, etc.
- **Audio**: MP3, WAV, etc.
- **URLs**: YouTube links, direct media URLs

**Max file size**: 50MB per file

### Upload Methods

#### 1. **Drag & Drop**
- Simply drag files from your computer/Dropbox into the upload area
- Supports multiple files at once

#### 2. **File Browser**
- Click "Select Files" to browse your computer
- Select multiple files

#### 3. **URL Ingestion**
- Click "Add URL" to ingest media from:
  - YouTube links
  - Direct image/video URLs
  - Any publicly accessible media

### Setting Up Your Media

For each file, you can configure:

- **Title**: Name your media
- **Source**: 
  - `grok` - For Grok-generated content
  - `sora` - For Sora-generated videos
  - `camera` - For camera captures
  - `other` - For everything else
- **Prompt**: The AI prompt used (if applicable)
- **Model**: The model used (e.g., "grok-2", "sora-v1")
- **Rights**: Owned, Licensed, or Unknown
- **Rating**: A (Premium), B (Standard), C (Basic)
- **Tags**: Add custom tags (press Enter to add)
- **Collections**: Select from:
  - `dreamnet` - General DreamNet content
  - `seed` - Seed content for the ecosystem
  - `ecosystem` - Ecosystem showcase
  - `agents` - Agent-related content
  - `starbridge` - StarBridge system content

### Earning Rewards

**Connect your wallet** to earn DREAM tokens for each upload! The system automatically grants rewards for media uploads.

### What Happens After Upload

1. **Processing**: Media is processed and stored in the Media Vault
2. **Thumbnails**: Automatically generated for images
3. **Metadata**: Extracted and indexed
4. **Spore Creation**: A Media Spore is created for tracking
5. **Rewards**: DREAM tokens granted (if wallet connected)
6. **Availability**: Media becomes searchable and usable across DreamNet

### Collections for Your Content

Based on your question about seeding "dream star" content:

- **`seed`** - Perfect for initial content seeding
- **`ecosystem`** - For showcasing the living ecosystem
- **`agents`** - For agent-related videos/content
- **`starbridge`** - For StarBridge system content

### Batch Upload Tips

1. Upload multiple files at once
2. Configure metadata for each file
3. Use "Upload All" to process everything
4. Files upload sequentially to avoid rate limiting

### YouTube Music

For YouTube music:
1. Use "Add URL" button
2. Paste the YouTube URL
3. Set source to "other" (or create a custom tag)
4. Add tags like "music", "youtube", etc.

### Your Previous Uploads

**Note**: Previous YouTube music and other media you've uploaded should still be stored in the Media Vault database. The system uses file hashing to prevent duplicates, so re-uploading the same file won't create duplicates.
